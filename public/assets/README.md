# Assets Directory

Place all image assets here:
- **Logo** — `logo.png` (+ animated variants)
- **Icons** — App icons, favicons
- **Illustrations** — Marketing & landing page images

## Usage

Reference in code:
```tsx
<img src="/assets/logo.png" alt="Synaptic" />
```

Or in CSS:
```css
background-image: url('/assets/logo.png');
```
